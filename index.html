Projetos em Ciência de Dados

Predição de vendas em rede de farmácias multinacional

Neste projeto foram usados Python, Estatística e técnicas supervisionadas de Machine Learning para realizar uma previsão das vendas nas próximas 6 semanas em todas as lojas de uma franquia de farmácias. Um Bot no Telegram foi construído para informar os resultados das previsões em tempo real. O resultado dessa solução, caso implementada, seria de 284M de dólares de receita para a franquia.

Análise de melhores preços para venda de produtos em e-commerce

Extração de dados com BeautifulSoup e análise exporatória de dados em Python para definição do melhor preço para vendas dos produtos de uma empresa emergente no ramo de e-commerce.

Identificação de imóveis para compra e revenda a fim de maximizar lucro

Análise de imóveis abaixo do preço médio de venda e definição do preço ideal para revenda a partir de análise exploratória de dados em Python.
